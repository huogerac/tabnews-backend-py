# tabnews-backend-py

This is a python backend side project of tabnews.com.br

### Para entender esta estrutura

- Iniciar pelos testes
```
tests
├── api
│   ├── test_api_create_tabnews.py
│   ├── test_api_list_tabnews.py
│   ├── test_api_login.py
│   └── test_api.py
├── database
│   └── test_db.py
└── services
    └── test_list_tabnews.py
```
